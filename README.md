# Java-Basics 

This repository contain simple program and examples to learn Java Basics. 
It is perfect for beginners who are starting their programming journey.


1. **Install VS Code**(Setup / “Build”)


   Go to: https://code.visualstudio.com
Download for Windows
Run the installer:
Click Next → Next → Install
 Select:
“Add to PATH”
“Open with Code” (optional but useful)

3. Open VS Code
Double-click the VS Code icon
OR
Press Win + S → type VS Code → open it

4. Install Required Extensions

Java Extension Pack (if using Java)

4. Install Language (Example: Java)

To run Java programs, install Java Development Kit:

Download JDK
Install it
Check in terminal:
java -version
